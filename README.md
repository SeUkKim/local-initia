# local-initia

this is not stable yet.

## Prerequisites
- [Docker](https://www.docker.com/products/docker-desktop/)

## Install local-initia

1. Run the following commands:

```sh
$ git clone git@github.com:initia-labs/local-initia.git
$ cd local-initia
```

## Start local-initia

- Start local-initia

```sh
$ docker compose up
```
